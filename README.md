# AI Career Predictor

## Overview
**AI Career Predictor** is a web-based application that predicts the most suitable career for a person based on their skills in **Programming, Mathematics, Data Analysis, Communication, and Creativity**.  

It uses a **Decision Tree Machine Learning model** trained on sample data and is deployed online so anyone can access it via a web browser.

---

## Features
- Predicts careers based on **5 skill inputs**.
- User-friendly **web interface** with a modern design.
- **Responsive design** works on desktop and mobile.
- Fully online using **Flask** backend.
- **Deployed on Render** for public access.

---

## Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Python, Flask
- **Machine Learning:** Scikit-learn, Pandas, NumPy
- **Deployment:** Render.com
- **Model Persistence:** Joblib

---

## How to Use
1. Open the live website:  
   [AI Career Predictor](https://ai--career-predictor.onrender.com)

2. Enter your skill ratings (1–5) for:
   - Programming  
   - Mathematics  
   - Data Analysis  
   - Communication  
   - Creativity

3. Click **Predict Career**.

4. The predicted career will appear below the form.
